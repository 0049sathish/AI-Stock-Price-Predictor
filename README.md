# AI Stock Price Predictor 📈

An AI-powered web application that predicts stock prices using Machine Learning and Deep Learning models.

## Features
- Real-time stock data from Yahoo Finance
- Multiple ML & DL models
- Interactive visualizations
- Multi-stock comparison

## Models Used
- Linear Regression
- Random Forest
- LSTM
- GRU

## Tech Stack
Python  
Streamlit  
TensorFlow  
Scikit-learn  
Plotly  
Yahoo Finance API

## Project Architecture

User → Streamlit UI → Feature Engineering → ML/DL Models → Prediction → Visualization
